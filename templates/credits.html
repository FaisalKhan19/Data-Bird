About Data Bird

Data Bird is tool designed to help you gather, track and visualize data.

Features

Currently there are 3 features available.

    Data Scraping Wizard: Easily scrape data from websites

    Price Tracking: Monitor prices of your favorite products

    API <-> Data: Directly get data from the endpoint and then Visualize it or save it as json or csv.
